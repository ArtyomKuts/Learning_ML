# Learning_ML

Этот репозиторий содержит эксперименты с алгоритмами машинного обучения. Здесь собраны мои учебные проекты и ноутбуки с практикой. Целью этих экспериментов является закрепление теоретических знаний на практике, изучение библиотек (scikit-learn, pandas, matplotlib) и исследование поведения ML-моделей.
